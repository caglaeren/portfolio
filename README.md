# 💜 Tuğba Çağla EREN — Portfolio

Kişisel portfolyo web sitesi. **Next.js**, **Tailwind CSS** ve **Framer Motion** ile geliştirildi. Mor gradyan teması, İngilizce/Türkçe dil desteği ve akıcı kaydırma animasyonları içeriyor.

🌐 **Canlı:** [caglaeren.vercel.app](https://caglaeren.vercel.app)

---

## ✨ Özellikler

- **Mor gradyan teması** — animasyonlu organik blob arka planlar
- **Çift dil desteği** — İngilizce / Türkçe geçiş butonu
- **Framer Motion** — kaydırmada yükselen/yüzen animasyonlar
- **Responsive tasarım** — mobil uyumlu, alt navigasyon barı
- **Bölümler:** Hero, Ben Kimim, Zaman Çizelgesi, Projeler, Teknoloji Yığını, İletişim, Footer
- **CV İndir** butonu
- **Ç favicon** — mor-pembe gradyanlı özel favicon

## 🚀 Kurulum

### Gereksinimler

- Node.js 18+
- npm veya yarn

### Adımlar

**Repoyu klonla:**

```bash
git clone https://github.com/caglaeren/portfolio.git
cd portfolio
```

**Bağımlılıkları yükle:**
```bash
npm install
```

**Geliştirme sunucusunu başlat:**
```bash
npm run dev
```

**Prodüksiyon Build**
```bash
npm run build
npm start
```

## 📄 Sitedeki Bölümler

| **Hero** | İsim, slogan, sosyal medya linkleri, CV indirme ve iletişim butonları |
